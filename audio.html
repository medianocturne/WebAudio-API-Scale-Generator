Steve's Web Audio API Scale Generator

bpm

Generates major, minor, melodic minor and harmonic minor scales in all 12 keys. Simply select your key and scale from the dropdown menus, as well scale direction and tempo.

Audio is generated 'in browser' using substractive synthesis via the WebAudio API, and is webKit and iOS compatible.


    Planned features include:
  • modes for all scales
  • delayed start
  • multiple octaves
  • ascending and descending
  • on screen note names
